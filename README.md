# Heropedia — Content Repository

This repository holds community-contributed role definitions for
[Heropedia.org](https://heropedia.org) — a library of historical heroes
playing professional roles, for use in LLM prompts.

Each Markdown file is one hero × role pairing. Files live under
their profession folder, e.g. product/product-critic-steve-jobs.md.

Content is MIT-licensed. Submissions happen through the Heropedia web
app; this repo is the source of truth for every file the site serves.
